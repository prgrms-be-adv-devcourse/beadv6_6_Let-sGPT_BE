# OFFICE_PRODUCTIVITY — 엑셀·스프레드시트 활용

엑셀이나 스프레드시트는 관리자 챗봇이 제공한 비식별 집계를 후속 정리하고, 피벗 분석·차트·교대 인수인계·정기 보고서에 활용할 수 있다. 현재 플랫폼에 자동 엑셀 내보내기 기능이 구현됐다고 가정하지 않는다.

권장 구성:

- 원본 탭: 조회 기간, 조회 시각, 지표, 그룹, 값, 데이터 출처를 수정 없이 보관
- 분석 탭: 피벗, 전일·전주 비교, 비율과 추이 계산
- 보고 탭: 핵심 변화, 확인이 필요한 상태, 담당자와 후속 조치
- 기준 탭: 상태값 의미, 기간의 시작 포함·종료 미포함 규칙, 시간대

주문 상태 분포와 추이는 결제 대기·실패 증가를 보는 데, 드롭 상태와 상품 완성도 집계는 오픈 준비 점검에, 회원 역할·가입 경로 집계는 접근 흐름 변화를 보는 데 적합하다. 서로 다른 조회 시각이나 기간 기준의 숫자를 같은 표에서 직접 비교하지 않는다.

회원 개인정보, 인증 토큰, 내부 UUID, 결제 민감정보, 임의 원본 행은 파일에 넣지 않는다. 특정 개인을 식별할 수 없는 승인된 집계만 사용한다. 반복 자동화가 필요하면 수동 복사에 의존하기 전에 승인된 집계 API나 내보내기 계약, 파일 보관 기간과 접근 권한을 먼저 정한다.
