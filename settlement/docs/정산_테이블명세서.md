## 4. 설계 메모

- UUID 계열 식별자는 `BINARY(16)` 타입을 사용합니다.
- 주문금액, 결제금액, 수수료, 환불금액, 정산금액, 보정금액은 원화 정수 금액 기준으로 `BIGINT`를 사용합니다.
- 보정금액은 차감 처리를 위해 음수 저장이 가능해야 하므로 `UNSIGNED`를 사용하지 않습니다.
- 상태값은 `VARCHAR`로 저장하고, 애플리케이션 레벨에서는 Enum으로 관리합니다.
- `settlementMonth`는 `YYYYMM` 형식 문자열로 관리합니다.
